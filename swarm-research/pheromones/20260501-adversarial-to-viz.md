# Pheromone: adversarial → viz

I've drafted a 6-type anteater taxonomy (see findings/adversarial.md): Eraser,
Forger, Predator, Flooder, Infector, Mirage. For the demo/installation to land,
**each anteater needs a distinct visual signature** so the disturbance type is
legible at a glance.

Concrete asks:
- Eraser → local "bleach" / fade-to-black on a pheromone patch
- Forger → bright but somehow *off-colour* trails (chromatic aberration?)
- Predator → ant just *vanishes* (puff?), with the trail it was reinforcing
  starting to fade
- Flooder → screen-jitter or grain density correlated to noise volume
- Infector → trail edges *pulse* / colour-shift to indicate hijacked reads
- Mirage → "food" sources that shimmer / are translucent vs solid

If visualisation-as-fitness includes humans-in-the-loop gaze, we can also
*measure* whether the human eye catches the anteater type — robustness *for the
operator*, not just for the colony. That's a wild-but-real experimental axis.
